# Risk Management App

A simple web application for risk management.

## Features

- Input risk factors with probability and impact.
- Calculate risk scores.
- Display risk analysis results in a bar chart.
- Export risk analysis to PDF.

## Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js
- jsPDF
